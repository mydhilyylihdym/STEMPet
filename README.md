# STEMPET

## Overview
STEMPET is an interactive embedded learning system designed to make STEM education more engaging through real-time user interaction and gamified learning techniques.

Built using the ESP32 microcontroller, the system combines embedded hardware and firmware to create a standalone educational platform with instant feedback and level-based progression.

---

## Features
- Standalone embedded learning system
- Real-time answer validation
- Instant visual feedback
- Streak and level progression
- Push-button based interaction
- TFT display interface
- WiFi-enabled web control
- Gamified learning experience

---

---

## System Working
The user interacts with the system using physical push buttons.  
Questions and visual feedback are displayed on the TFT display in real time.

The system validates user responses instantly and tracks:
- Correct answer streaks
- Level progression
- Learning interaction flow

A web-based WiFi interface is also implemented for system control and interaction.
